## Problem

Students and campus visitors often notice animal activity around campus, but there is no simple way to document sightings, share locations, and keep a record of where certain animals have been seen over time.

## Approach

Developed a full-stack web and mobile application using the MERN stack with TypeScript to allow users to create and browse animal sighting posts. Designed features for submitting sightings with location data, uploaded photos, captions, and animal type information.
Built the application around a structured post system so users could record sightings in a consistent format and explore reports from others. Focused on creating a practical and user-friendly interface for location-based community reporting.

## Outcome

Created a functional tracking platform that combined full-stack application development with location-aware features and media upload support. The project strengthened experience in TypeScript, MongoDB data modeling, and building user-facing applications that connect real-world activity to interactive digital tools.
